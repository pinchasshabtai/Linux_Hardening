###תיאור:
תוכנה זאת משמשת להתיר אפליקציות ותוכנות וסקריפטים על פי סריקה (white list) ולחסום את כל השאר ב - Linux Ubuntu 24.04

### הערה 1: 
יש להריץ את כל הפקודות ב - sudo.

### הערה 2:
אם הקבצים הגיעו מ - Windows ייתכן ויהיה צורך להעתיק הטקסט ידנית ולא פשוט העתק הדבק לקובץ.

### התקנה:
להעתיק את כל הקבצים לתיקייה במחשב ולהריץ את הפקודה install_fapolicyd.sh
מעכשיו המחשב נכנס למצב סריקה וכל מה שתשתמשו בו יהיה זמין (יש צורך גם בכיבוי, נעילה, וכו).

### פעולות:
כדי לעצור את הסריקה ולהתחיל את האכיפה יש להריץ את הפקודה stop-sf
כדי להוסיף אפליקציה\סקריפט ל - white list יש להריץ את הפקודה start-sf ואז להשתמש באפליקציה ובסיום להריץ את הפקודה stop-sf

### מידע: 
יש ארבע קבצים של חוקים בנתיב /etc/fapolicyd/rules.d/
01deny-my-rule.rules
02scan.rules
03allow-my-rule.rules
04deny-all.rules
חוקים מתירים שלך תוסיף לקובץ 03
חוקים אוסרים שלך תוסיף לקובץ 01
אזהרה!!!! אל תמחק או תשנה את הקבצים 02 או 04 זה ישבור החסימה ועלול לשבור את כל מערכת ההפעלה!!
אזהרה!!!! אל תמחק או תשנה את החוקים הקיימים בקבצים 01 או 03 זה ישבור את החסימה ועלול לשבור את כל מערכת ההפעלה!!
